# Report Synthesis Instructions

## Report Structure
Write these sections using `write_report_section()`:

### 1. Executive Summary
- Total number of anomalies found
- Total variance explained
- Top 3 findings by materiality
- Overall data quality assessment

### 2. Methodology
- Brief description of the analysis approach
- Tables examined
- Time periods covered
- Materiality thresholds applied

### 3. Findings by Category
Group findings by category:
- COGS anomalies
- Revenue anomalies
- FX anomalies
- Budget misalignments
- Classification errors

For each finding, reference the finding ID and include:
- Root cause summary
- Key evidence
- Impact quantification
- Confidence level

### 4. Recommendations
Prioritized list of corrective actions:
- Immediate fixes (data corrections)
- Process improvements
- Monitoring suggestions

### 5. Data Quality Notes
- Any data quality issues discovered
- Missing data or gaps
- Limitations of the analysis

## Writing Style
- Be specific: "COGS increased 25% in CC-300" not "costs went up"
- Quantify everything: amounts, percentages, periods
- Reference finding IDs: "As detailed in F-001..."
- Use professional financial analysis tone
